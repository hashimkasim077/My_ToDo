# 📝 **MyToDo — The Task Manager**

_A super clean Flutter to‑do app with full CRUD and zero complexity._

---

## 🌈 What Is This?
**MyToDo** is a **one‑file Flutter app** that shows how to build a real task manager with:
✔ Create  
✔ Read  
✔ Update  
✔ Delete

All stored locally using **Hive** — no backend, no heavy libs, no chaos.

---

## ⚡ Why It’s Awesome
- ✍️ Add tasks with **due dates**
- ⏰ Set **reminders**
- 🔁 Auto **recurring tasks** (daily / weekly / monthly)
- 🧠 **Smooth UI updates** with StatefulWidget
- 🗑️ Long‑press to delete
- 📦 No bloated architecture — ALL in one file
- 🧪 Perfect for learning Flutter + Hive

---

## 📦 Tech Stack
- 🐦 Flutter  
- 💛 Dart  
- 🐝 Hive (fast local database)  
- 🪩 Material Design UI  

---

## 🚀 How It Works
1. Open the app  
2. Add a task  
3. Edit tasks by tapping them  
4. Long‑press to delete  
5. Tasks saved forever locally with Hive
